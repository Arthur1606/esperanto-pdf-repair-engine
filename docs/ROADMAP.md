# TEKIRA Roadmap

Este es el mapa de evolución oficial de TEKIRA hacia la versión 3.0.

## Estado Actual (v2.3)
- **100% Offline**
- 12 Motores Especializados
- 2,201 reparaciones exitosas
- <0.3% de errores residuales
- 0 Overlays visuales garantizados

## Objetivos a Corto Plazo (v2.5)
- **Integración Nativa de OCR:** Permitir a TEKIRA leer PDFs que son puramente imágenes.
- **Soporte de Tablas Complejas:** Mejorar `Arĥitekto` para reconstruir tablas sin dañar las líneas de celda.
- **Expansión de Biblioteko:** Integrar Wikipedia Esperanto completa (15GB de texto).

## Objetivos a Largo Plazo (TEKIRA v3.0)
- **Modelo de Lenguaje (LLM) Offline:** Implementar un SLM (Small Language Model) de ~3B parámetros especializado en Esperanto que corra estrictamente de forma local. Este LLM reemplazará a `Juĝanto` para entender contextos bilingües y lograr finalmente 0 revisiones humanas.
- **Generación Automática de Recursos:** Convertir PDFs rotos no solo a PDFs reparados, sino extraerlos limpiamente a ePub y HTML.
